About This Demo

This is a fictional astronomy door built to demonstrate Easy FYN.

Big Sky Backyard Astronomy is not a real club, organization, event calendar, or member network. It was created to show what a current FYN site can honestly do.

The site contains a home page, a creator-defined links page, a categorized resources page, and this about page. Everything is plain static HTML and CSS. There is no login, database, analytics script, form submission, feed, search service, recommendation engine, or central community registry.

A real creator could use the same structure to make an existing astronomy group easier to recognize and find, while pointing visitors to a club website, mailing list, public event calendar, Discord server, forum, or other place the community already uses.

A door, not the room

FYN does not attempt to replace the community. It gives the community one clear, quiet place to explain itself and point outward.

No tracking. No data collected. Static site generated by Finding Your Neighborhood.
